# chatRobot
调用百度语音和图灵机器人API

需要在Robot.java里修改成自己的图灵机器人密钥，
在FirstTryActivity.java，Second.java，Third.java，Fourth.java里修改成自己的百度语音密钥。
（这几个文件里其实有很多重复的代码，以后有时间再重构下）
